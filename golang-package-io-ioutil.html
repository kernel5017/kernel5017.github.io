io/ioutil

import “io/ioutil”

func ReadAll

读取数据使用

ReadAll返回读取到的所有数据，直到遇到error或者EOF为止。如果执行成功的话，应该是err == nil，而不是err == EOF。因为ReadAll是从开头读到EOF,它不会将读到的EOF作为一个error。

func ReadDir

读取目录，返回目录下文件名和目录名

ReadDir读取给定目录，返回指定目录下所有目录和文件的名字列表；注意它不会对目录进行遍历，比如目录test下有目录test1，他只会列出test目录，而不会列出test1目录。

func ReadFile

读取文件，返回文件所有内容

读取给定文件并返回文件内所有内容。执行成功时，应该是err == nil,而不是err == EOF

func TmepDir

TempDir会在给定目录下创建一个新的临时目录。目录名字根据pattern自动生成，会在你指定的pattern后面添加一串随机的字符串。如果pattern包含”*‘’符号，那么会用随机字符串替换最后一个“星号”。TempDir会返回临时目录的名字。如果你不指定dir参数，那么TempDir会使用默认的os.TmepDir作为参数。

举例：

func TempFile

TempFile根据pattern在dir目录下生成临时文件，并打开这个文件提供读写，返回*os.File。临时文件的名字根据pattern生成，会在你指定的pattern后添加一串随机字符串，如果pattern中有“星号”，那么会用随机字符串替换最后一个星号。如果没有指定dir，那么使用默认的os.TmepDir目录作为dir参数。

举例：

func WriteFile

WriteFile会向给定的filename文件中写入数据。如果这个文件不存在，那么WriteFile将会创建这个文件；如果文件存在，那么WriteFile将会在写入前将这个文件清空，不会改变这个文件的权限。
